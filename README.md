## Overview
This repository contains the submissions for **Unit 1: Generative AI & NLP Hands-On**.  
The objective of this unit is to understand **core Generative AI concepts**, explore **Transformer-based model architectures**, and analyze how different models behave when applied to various NLP tasks.

The work focuses on **observation, experimentation, and architectural reasoning**, rather than model accuracy or training.

## Key Concepts Covered

- Difference between **Encoder-only** and **Encoder-Decoder** architectures
- Understanding why **BERT and RoBERTa fail at text generation**
- Importance of **model-task alignment**
- Role of **fine-tuning** in Question Answering tasks
- Practical use of **Hugging Face pipelines**
